# kiwimaddog2020.github.io

The hub site for my research notes (https://kiwimaddog2020.github.io/).
Each note is a GitHub Pages site published from the repository it
describes; this repo only holds the index.

The Notes list between the `notes:start` / `notes:end` markers in
`index.md` is generated by `bin/build_index.py`, which lists my public
repositories tagged with the `research-note` topic and uses each repo's
description and homepage. A weekly GitHub Actions run keeps it fresh;
entries can also be edited by hand between rebuilds.

Run locally: `python3 bin/build_index.py` (uses `GITHUB_TOKEN` when set,
anonymous otherwise). Tests: `python3 -m pytest tests/`.
